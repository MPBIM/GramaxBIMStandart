title: 3.2 Шаблоны проектов
---

Корпоративный шаблон проекта представляет из себя предварительно настроенный сотрудником BIM-отдела шаблонный файл формата "RVT". Данные файлы расположены по пути: **M:\\BIM\\03\_Шаблоны**

Шаблоны проекта могут выступать в качестве хранилища системных семейств, которые нельзя сохранить в виде отдельных файлов.

:::note 

Работа в исходном файле шаблона проекта строго запрещена! Проектировщику необходимо начинать работу с моделью только в полученном от сотрудника BIM-отдела файле.

:::

Основные принципы создания шаблонов проектов Revit:

1. Шаблоны проектов разделены на разделы проектирования и версии приложения Revit;

2. Шаблоны содержат минимальный необходимый для работы набор наиболее используемых системных или загружаемых (п. 3.4 настоящего стандарта) семейств;

3. Шаблоны содержат минимальный необходимый для работы набор параметров проекта и общих параметров из корпоративного файла общих параметров (п. 3.3 настоящего стандарта);

4. Шаблон содержит минимальный необходимый для работы набор преднастроенных видов, спецификаций и листов, а также их шаблонов;

5. Виды, спецификации, листы шаблона проекта сгруппированы согласно корпоративным правилам организации диспетчера проекта (п. 4.4 настоящего стандарта);

6. Шаблоны содержат минимальный необходимый для работы набор типоразмеров линий, штриховок и размеров.
