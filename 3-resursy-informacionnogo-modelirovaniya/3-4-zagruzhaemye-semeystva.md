---
order: 4
title: 3.4 Загружаемые семейства
---

Работа с корпоративными загружаемыми семействами и их библиотеками осуществляется только сотрудниками BIM-отдела.

Применяемые для информационного моделирования загружаемые семейства должны соответствовать следующим критериям:

-  именоваться согласно корпоративным правилам наименования (п. 4.5.3-5 настоящего стандарта);

-  обладать необходимой для дальнейшей работы параметризацией;

-  содержать только корпоративные общие параметры (п. 3.3 настоящего стандарта);

-  иметь необходимый для передачи минимальных визуальных характеристик уровень детализации.

**Библиотека семейств** представляет из себя хранилище всех доступных для применения загружаемых семейств на сетевом диске и представлена 2-мя областями: центральной библиотекой и локальной библиотекой проекта.

Центральная библиотека семейств расположена по пути **M:\\BIM\\01\_Библиотека семейств** и представляет из себя хранилище общеприменимых для всех проектов загружаемых семейств.

Локальная библиотека семейств проекта расположена в составе каждого проекта по пути **M:\\BIM\\00\_Проекты\\Имя проекта\\11\_Библиотека\\01\_Семейства** и представляет из себя хранилище:

-  уникальных для данного проекта семейств;

-  временных семейств, требующих утверждения в проекте сотрудниками BIM-отдела.

Для дальнейшего применения семейств из локальной библиотеки в других проектах необходимо осуществить их проверку на предмет соответствия требованиям настоящего стандарта и при успешном ее прохождении переместить их в локальную библиотеку другого проекта или центральную библиотеку.

:::note 

Использовать семейства из иных источников (создавать загружаемые семейства самостоятельно, брать из других проектов) не допускается, т.к. они могут перегружать модель, содержать ошибки, дублироваться с имеющимися (с добавлением числового индекса в конец имени), некорректно специфицироваться, приносить с собой сторонние общие параметры, вызывать проблемы при передаче модели в экспертизу и т.д..

:::
