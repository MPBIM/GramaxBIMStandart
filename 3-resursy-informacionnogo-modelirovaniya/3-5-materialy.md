## Требования к материалам

Применяемым для информационного моделирования материалы должны соответствовать следующим критериям:

-  именоваться согласно корпоративным правилам наименования ([п. 4.5.9 настоящего стандарта](./../../BIMStandart/requirements/naming/material));

-  содержать только корпоративные общие параметры ([п. 3.3 настоящего стандарта](./../../BIMStandart/resourses/shared-parameters));

-  применяемые для визуализации материала пользовательские текстуры должны быть расположены в среде общих данных на том же сетевом диске, что и библиотека материалов.

-  обладать необходимой для дальнейшей работы параметризацией (согласно описанным ниже требованиям);

### Параметры материалов

Каждый материал проекта должен содержать значения обязательного минимума параметров:

-  "Имя" – системный параметр, значение которого соответствует полному наименованию материала;

-  "ADSK\_Материал наименования" – общий параметр, который содержит информацию из поля характеристики материала ([п. 4.5.9 настоящего стандарта](./../../BIMStandart/requirements/naming/material));

-  "ADSK\_Материал тип подсчета" – общий параметр для указания типа подсчета материала.

По необходимости для материалов вводятся дополнительные параметры:

-  "ADSK\_Обозначение" – общий параметр для указания нормативного документа материала;

-  "ADSK\_Норма расхода" – общий параметр для указания коэффициента запаса/нахлеста материала;

-  "ADSK\_Единица измерения" – общий параметр для указания единицы измерения материала.

-  "Описание" – системный параметр, который содержит расшифровку индекса материала из таблицы индексов материалов ([п. 4.5.9 настоящего стандарта](./../../BIMStandart/requirements/naming/material));

-  "Класс" – системный параметр, значение которого соответствует типу материала в таблице индексов материалов ([п. 4.5.9 настоящего стандарта](./../../BIMStandart/requirements/naming/material));

## Библиотека материалов

Центральная библиотека материалов расположена по пути **M:\\BIM\\02\_Материалы** и представляет из себя файл формата **rvt**, содержащий минимальный исходный список корпоративных материалов. Пользовательские текстуры библиотеки материалов располагаются по пути **M:\\BIM\\02\_Материалы/Текстуры**

Локальная библиотека семейств проекта  вводится по необходимости и расположена в составе каждого проекта по пути **M:\\BIM\\00\_Проекты\\Имя проекта\\11\_Библиотека\\02\_Материалы** и представляет из себя хранилище уникальных для данного проекта материалов. Пользовательские текстуры библиотеки материалов располагаются по пути **M:\\BIM\\00\_Проекты\\Имя проекта\\00_BIM\\11\_Библиотека\\03\_Текстуры**
