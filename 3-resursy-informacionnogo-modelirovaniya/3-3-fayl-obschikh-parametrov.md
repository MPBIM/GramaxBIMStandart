order: 3
title: 3.3 Файл общих параметров
---

Файл общих параметров (ФОП) – это текстовый документ со строго зарегламентированной структурой, который содержит основные свойства общих параметров, так называемые определения параметров (имя, категория параметра, тип данных, подсказка, видимость и возможность редактирования пользователем).

В рамках информационного моделирования компании используется файл **МП4\_ФОП**, расположенный по пути: **M:\\BIM\\04\_ФОП**

:::note 

Проектировщикам строго запрещено вносить в данный файл изменения, создавать в нем новые общие параметры или пользоваться другими файлами общих параметров!

:::

Для идентификации корпоративных общих параметров первое поле имени общего параметра принимает значение "ADSK" (параметры исходных шаблонов ADSK) или "ОП*"* (сформированные сотрудниками BIM-отдела корпоративные общие параметры).
