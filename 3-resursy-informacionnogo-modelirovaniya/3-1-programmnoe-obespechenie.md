---
order: 1
title: 3.1 Программное обеспечение
---

Настоящий стандарт предусматривает в качестве основного программного обеспечения для информационного моделирования нижеуказанные приложения.

**Для сотрудников проектного отдела:**

-  Autodesk Revit 2023 русской языковой версии;

-  Autodesk AutoCAD 2023 русской языковой версии;

-  Autodesk Civil 3D 2023 русской языковой версии.

:::info 

Любой случай обновления вышеуказанного программного обеспечения проектировщиков должен быть рассмотрен и утвержден BIM-менеджером/координатором.

:::

**Дополнительно для сотрудников BIM-отдела:**

-  Autodesk Navisworks Manage 2023 русской языковой версии;

-  Visual Studio (по запросу для BIM-разработчиков);

-  Visual Studio Code (по запросу для BIM-разработчиков);

-  Notepad++ (по запросу для BIM-разработчиков).

:::info 

Переход на новые версии программного обеспечения осуществляется в соответствии с корпоративной BIM-стратегией.

:::
