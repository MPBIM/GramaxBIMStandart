title: Приложение Д "Виды для экспорта из Revit"
---

Для осуществления экспорта информационной модели в другие форматы **в каждой модели проекта**  должны быть предусмотрены следующие предварительно настроенные 3D-виды:

1. **Navisworks**  – вид, который предназначен для экспорта модели в формат nwc для последующих проверок на коллизии или для сборки сводной модели. 

:::info 

Наименование данного вида должно содержать слово “Navisworks”, при этом в проекте должен быть только один вид, содержащий в имени слово “Navisworks”.

:::

1. **IFC** – вид, который применяется для экспорта в формат ifc, для последующей передачи в сторонние программы, заказчику или экспертизу.

Данные виды используются исключительно специалистами BIM-отдела и располагаются в группировке диспетчера проекта "00_BIM", которая скрыта организационной схемой диспетчера проекта.

[image:./image (1).png:::0,0,100,100:::center]

:::note 

Проектировщикам запрещено вносить в данные виды какие-либо изменения.

:::

К вышеуказанным видам предъявляются следующие требования:

-  виды должны отображать все относящиеся к текущему разделу элементы информационной модели;

-  виды не должны отображать следующие элементы:

   -  аннотационные элементы (уровни, оси, линии модели и др.);

   -  элементы заданий смежным разделам;

   -  дубликаты смежных разделов;

   -  формы (за исключением строительного объема для вида IFC).
