Среда общих данных является единым источником достоверной и согласованной  информации для всех участников информационного моделирования и обеспечивает единую рабочую  среду, позволяющую осуществлять контроль проектной информации и ее совместное  использование всеми участниками многодисциплинарной проектной группы.

Среда общих данных для осуществления информационного моделирования представлена несколькими областями:

1. **Revit Server** – содержит информационные модели проектов с расширением ".rvt";

2. **Папка BIM** – общая для всех проектов папка ресурсов информационного моделирования на соответствующем сетевом диске;

3. **Папка 00.BIM**  – папка локальных ресурсов информационного моделирования, которая расположена в составе каждого проекта.

:::note 

Локальные диски пользователей не обеспечивают совместного доступа к информации всеми участниками и поэтому не являются средой общих данных. При осуществлении информационного моделирования **запрещается** хранить и использовать необходимую для выдачи документации и передачи модели информацию (информационные модели, подложки, изображения и т.д.) на локальных дисках.

:::

## Revit Server

Revit Server – это серверное приложение, которое является основным приложением для совместной работы над проектами Revit.

Деление проектов и соответствующих им информационных моделей на Revit Server выполняется по следующим установленным в компании принципам:

-  для размещения информационных моделей проектов жилых зданий выделяются сервера с именами "Living1", "Living2" и т.д.;

-  для размещения информационных моделей проектов гражданских объектов выделяются сервера с именами "Civil1", "Civil2" и т.д.;

-  количество проектов на сервере определяется сотрудниками BIM-отдела на основании анализа загруженности данного сервера, а количество серверов определяется количеством разрабатываемых проектов;

-  для осуществления эффективной работы некоторые из проектов могут быть перемещены сотрудниками BIM-отдела на другие сервера.

## Центральная библиотека BIM-ресурсов

Папка ресурсов информационного моделирования расположена по следующему пути: **M:\\BIM**

Файловая структура папки "BIM" соответствует следующей организационной схеме:

[image:./file.excalidraw (5).svg:::0,0,100,100:::center]

:::note

:::
